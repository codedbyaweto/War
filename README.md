# War Card Duel

War Card Duel is a lightweight card game built with HTML, CSS, and JavaScript.  
It uses the Deck of Cards API to fetch and draw real playing cards.  
Each round, two cards are drawnâ€”one for the computer and one for the player.  
The higher card wins the round and earns a point. When the deck is empty,  
the game announces the final winner.

---

## Features

- Fetches a freshly shuffled deck from an external API  
- Draws two cards per round  
- Automatically determines the winner based on card value  
- Tracks scores for both you and the computer  
- Displays remaining cards in the deck  
- Disables gameplay when the deck is empty  
- Declares the final winner at the end of the game  

---

## Technologies Used

- **JavaScript (ES6+)**
- **Deck of Cards API**
- **HTML & CSS**
- DOM manipulation
